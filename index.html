Push Pull Legs Workout Routine

The Push, Pull, Legs Workout Routine for Beginners

Monday - Push Day

Exercise Sets x Reps
Barbell Bench Press 4 x 8-10
Overhead Dumbbell Press 3 x 10-12
Incline Dumbbell Flyes 3 x 12-15
Triceps Dips 3 x 10-12
Lateral Raises 3 x 12-15

Tuesday - Pull Day

Exercise Sets x Reps
Deadlifts 4 x 6-8
Pull-Ups 3 x 8-10
Barbell Rows 3 x 8-10
Seated Cable Rows 3 x 10-12
Face Pulls 3 x 12-15

Wednesday - Legs Day

Exercise Sets x Reps
Squats 4 x 8-10
Leg Press 3 x 10-12
Lunges 3 x 10-12 per leg
Leg Curls 3 x 10-12
Calf Raises 4 x 12-15

Thursday - Push Day

Exercise Sets x Reps
Barbell Bench Press 4 x 8-10
Overhead Dumbbell Press 3 x 10-12
Incline Dumbbell Flyes 3 x 12-15
Triceps Dips 3 x 10-12
Lateral Raises 3 x 12-15

Friday - Pull Day

Exercise Sets x Reps
Deadlifts 4 x 6-8
Pull-Ups 3 x 8-10
Barbell Rows 3 x 8-10
Seated Cable Rows 3 x 10-12
Face Pulls 3 x 12-15
